# Margin
Sensical markdown first note taking app. Collaborative real-time at some point.

# Style
- snake_case for all code
- PascalCase for Classes, React Components, Types
- kebab-case for filenames
- Write components as a lambda
- Add semicolons everywhere applicable (including imports)
- Use tabs and tabs are 4 spaces

# Stack
- React 19 w/ compiler
- Convex
- shadcn
